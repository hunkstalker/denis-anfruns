# Sistema de Gestió de Tasques

Aplicació completa per gestió de tasques i projectes en equip amb funcions en temps real.

## Característiques

- Autenticació segura amb JWT
- Taulers Kanban interactius
- Actualitzacions en temps real amb WebSockets
- Sistema de notificacions
- Assignació de tasques i prioritats
- Estadístiques i informes

## Stack Tècnic

- Frontend: React amb Hooks i Context API
- Backend: Node.js amb Express
- Base de dades: MongoDB
- Temps real: Socket.io
