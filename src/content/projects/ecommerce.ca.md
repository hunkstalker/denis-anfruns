# E-Commerce Vintage

Botiga online completa per productes vintage amb totes les funcionalitats modernes.

## Característiques Principals

- Catàleg de productes amb filtres avançats
- Carret de compres persistent
- Integració amb Stripe per pagaments
- Panell d'administració
- Gestió d'inventari en temps real
- Sistema de ressenyes i valoracions

## Tecnologies

El projecte utilitza l'stack modern de Vue:
- Nuxt 3 per SSR i millor SEO
- PostgreSQL amb Prisma ORM
- Stripe per pagaments segurs
- TailwindCSS per disseny responsive
