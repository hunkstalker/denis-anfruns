# Portfolio Personal

Aquest és el meu portfolio personal desenvolupat amb tecnologies web modernes. Característiques principals:

- **Disseny Modern**: Utilitza glassmorphism i animacions subtils
- **Multiidioma**: Suport complet per ES, CA i EN
- **SEO Optimitzat**: Metadades i estructura semàntica
- **Responsive**: Funciona perfectament en tots els dispositius
- **Dark Mode**: Sistema de temes amb persistència local

## Tecnologies Utilitzades

El projecte està construït amb:
- Astro com a framework principal
- TypeScript per type safety
- TailwindCSS per estils
- React per components interactius

## Reptes i Aprenentatges

Durant el desenvolupament, vaig enfrontar diversos reptes interessants com la implementació d'un sistema de routing multiidioma que fos SEO-friendly i la creació d'efectes visuals premium mantenint el rendiment.
