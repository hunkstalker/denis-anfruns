# App Móvil de Fitness

Aplicación completa para tracking de fitness y salud disponible en iOS y Android.

## Funcionalidades

- Seguimiento de rutinas de ejercicio
- Contador de calorías y macros
- Temporizadores y cronómetros
- Estadísticas y gráficos de progreso
- Sincronización en la nube
- Modo offline

## Stack Tecnológico

- React Native con Expo
- Firebase (Auth, Firestore, Storage)
- Redux para gestión de estado
- Charts para visualización de datos
