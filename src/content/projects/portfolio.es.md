# Portfolio Personal

Este es mi portfolio personal desarrollado con tecnologías web modernas. Características principales:

- **Diseño Moderno**: Utiliza glassmorphism y animaciones sutiles
- **Multiidioma**: Soporte completo para ES, CA y EN
- **SEO Optimizado**: Metadatos y estructura semántica
- **Responsive**: Funciona perfectamente en todos los dispositivos
- **Dark Mode**: Sistema de temas con persistencia local

## Tecnologías Utilizadas

El proyecto está construido con:
- Astro como framework principal
- TypeScript para type safety
- TailwindCSS para estilos
- React para componentes interactivos

## Desafíos y Aprendizajes

Durante el desarrollo, enfrenté varios desafíos interesantes como la implementación de un sistema de routing multiidioma que fuera SEO-friendly y la creación de efectos visuales premium manteniendo el rendimiento.
