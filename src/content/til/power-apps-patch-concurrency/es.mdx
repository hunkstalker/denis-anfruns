# Función Patch y Concurrencia

Usa `If-Match` en headers o la configuración de concurrencia de la lista para evitar sobrescrituras accidentales.

```powerfx
Patch(Orders, ThisItem, { Status: "Approved" })
```

Esta es una nota rápida sobre una lección aprendida.
