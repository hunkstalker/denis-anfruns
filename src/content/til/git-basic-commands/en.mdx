### What each command does

- `git init`: Initializes a new Git repository in your current folder.
- `git add .`: Adds all modified files to the staging area.
- `git commit -m "..."`: Saves the committed changes with a descriptive message.
- `git branch -M main`: Renames the default branch to `main` (current standard).
- `git remote add origin`: Connects your local repository to the remote one (GitHub, GitLab, etc.).
- `git push -u origin main`: Uploads changes to the cloud and links your local branch with the remote one.
