### Què fa cada comanda

- **`git init`**: Inicialitza el repositori de git local.
- **`git add .`**: La paraula `add` serveix per afegir canvis a l'stage (preparació) i el punt `.` selecciona tots els fitxers.
- **`git commit -m "Initial commit"`**: Crea el primer commit. El flag `-m` permet afegir el missatge del commit directament.
- **`git branch -M main`**: Reanomena la branca actual a `main`.
    > **Nota**: Això és útil si la teva configuració per defecte crea la branca com a `master`.
- **`git remote add origin ...`**: Vincula el repositori local amb el remot. `origin` és l'àlies que farem servir per referir-nos a l'URL del repositori (https://github.com/hunkstalker/stardraw).
- **`git push -u origin main`**: Puja els canvis a GitHub. El flag `-u` vincula la teva branca local `main` amb la remota `origin/main`, permetent-te fer servir simplement `git push` en el futur.
