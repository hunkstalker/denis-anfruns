### Què fa cada comanda

- `git init`: Inicialitza un nou repositori de Git a la teva carpeta actual.
- `git add .`: Afegeix tots els fitxers modificats a l'àrea de preparació (stage).
- `git commit -m "..."`: Desa els canvis confirmats amb un missatge descriptiu.
- `git branch -M main`: Canvia el nom de la branca per defecte a `main` (estàndard actual).
- `git remote add origin`: Connecta el teu repositori local amb el remot (GitHub, GitLab, etc.).
- `git push -u origin main`: Puja els canvis al núvol i vincula la teva branca local amb la remota.
