### Qué hace cada comando

- `git init`: Inicializa un nuevo repositorio de Git en tu carpeta actual.
- `git add .`: Añade todos los archivos modificados al área de preparación (stage).
- `git commit -m "..."`: Guarda los cambios confirmados con un mensaje descriptivo.
- `git branch -M main`: Renombra la rama por defecto a `main` (estándar actual).
- `git remote add origin`: Conecta tu repositorio local con el remoto (GitHub, GitLab, etc.).
- `git push -u origin main`: Sube los cambios a la nube y vincula tu rama local con la remota.
