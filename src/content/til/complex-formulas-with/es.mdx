# Simplificando fórmulas con With

La función `With` permite definir variables locales dentro de una fórmula, haciendo el código más limpio y rápido.

```powerfx
With({ profit: Price - Cost }, If(profit > 0, "Gain", "Loss"))
```

Esta es una nota rápida sobre una lección aprendida.
