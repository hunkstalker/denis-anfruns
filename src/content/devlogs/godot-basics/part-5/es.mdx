# Matemáticas para matar marcianos

Hasta ahora, todo se movía en línea recta. Pero las naves enemigas molonas no hacen eso. Hacen elipses, "ochos" y picados.
Para eso necesitamos dos amigos: **El Seno y el Coseno**.

## ¿Qué vamos a aprender?
1.  **Trigonometría de Supervivencia**: Cómo usar `sin(time)` para hacer que los enemigos "floten" u ondulen.
2.  **Círculos**: Cómo usar `cos` y `sin` juntos para movimiento orbital.
3.  **Path2D**: La forma fácil. Dibujar curvas con el ratón (`PathFollow2D`) para rutas complejas.
4.  **Tweens**: Programar secuencias de movimiento (entrar, parar, salir).
